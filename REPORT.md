# Project Report – CP3402 Supplementary Assignment

## Overview

This project involved developing a WordPress website for U3A Townsville using a child theme and a modern web development workflow.

The website was designed to provide information about community activities, courses, membership, and contact details for older adults interested in lifelong learning and community participation.

## Website Design

The website was designed with a clean and accessible layout suitable for older adults.

The design focused on:

* Simple navigation
* Readable content
* Clear headings
* Consistent styling
* Community-focused imagery
* Goal-driven call-to-action buttons

---

## Development Workflow

The project used GitHub for version control with regular commits during development.

Docker was used to create a local development environment for WordPress.

The Astra Child theme was created to customise the appearance of the website while following WordPress development practices.
